# Flamel

This is an open-source reimplementation of _Alchemy_. It's a web app written in Vue.

## Project setup
```
yarn install
```

### Dev server
```
yarn serve
```

### Production build
```
yarn build
```

### Unit tests
```
yarn test:unit
```

### Lint
```
yarn lint
```

[Vue CLI configuration reference](https://cli.vuejs.org/config/).
